Inicio rapido

  1. Ejecuta devkit new mi-app --laravel para crear un proyecto
  2. Entra al directorio: cd ~/devkit-projects/mi-app
  3. Levanta el entorno: devkit start
  4. Abre https://mi-app.test en tu navegador

Comandos

devkit new <name> --laravel Crear proyecto Laravel
devkit new <name> --next Crear proyecto Next.js
devkit new <name> --nuxt Crear proyecto Nuxt
devkit new <name> --astro Crear proyecto Astro
devkit start Levantar entorno del proyecto actual
devkit stop Detener todos los servicios
devkit start --mysql Levantar con MySQL en Docker
devkit start --mailpit Levantar con Mailpit (correo)
devkit doctor Verificar dependencias
devkit list Ver proyectos registrados
devkit mail Abrir Mailpit en el navegador
