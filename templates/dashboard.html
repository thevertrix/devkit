Comandos Principales

devkit setupInstala Caddy, mkcert, Docker y crea wrappers de runtime vía Docker
devkit new <name> --laravelCrear proyecto Laravel
devkit new <name> --phpCrear esqueleto basico PHP
devkit new <name> --nextCrear proyecto Next.js
devkit new <name> --nuxtCrear proyecto Nuxt
devkit new <name> --angularCrear proyecto Angular
devkit new <name> --nestjsCrear proyecto NestJS
devkit new <name> --astroCrear proyecto Astro
devkit new <name> --svelteCrear proyecto SvelteKit
devkit startLevantar entorno — puerto auto-asignado si hay conflicto
devkit stopDetener todos los servicios y proxy
devkit stop --mysql --redisDetener servicios puntuales
devkit stop --php --node --pythonDetener runtimes puntuales
devkit doctorVerificar estado — muestra runtimes locales o vía Docker
devkit listVer proyectos registrados
devkit mailAbrir Mailpit en el navegador

Runtimes locales vía Docker

node -vEjecuta node en contenedor Docker (tras devkit setup)
npm installnpm también resuelve en contenedor
php -vPHP en contenedor — sin instalar localmente
composer installComposer en contenedor
python --versionPython en contenedor
pip install requestspip también resuelve en contenedor
devkit services --set --node=22Cambiar version y volver a correr devkit setup
devkit services --set --php=8.2Cambiar version PHP del wrapper

Runtimes en proyecto (devkit start)

devkit start --node=20Servidor dev del proyecto en contenedor Node 20
devkit start --php=8.3Servidor dev del proyecto en contenedor PHP 8.3
devkit start --python=3.12Contenedor Python listo en el puerto del proyecto
devkit stop --nodeDetener contenedor Node.js del proyecto
devkit stop --phpDetener contenedor PHP del proyecto
devkit stop --pythonDetener contenedor Python del proyecto
devkit services --set --node=20Version Node por defecto para nuevos proyectos
devkit services --set --python=3.12Version Python por defecto

Servicios Docker con versiones

devkit start --mysqlMySQL con version por defecto
devkit start --mysql=8.0MySQL 8.0
devkit start --postgres=15PostgreSQL 15
devkit start --redis=7Redis 7
devkit start --mailpitMailpit (servidor de correo)
devkit services --listVer versiones por defecto de servicios
devkit services --set --mysql=8.0Fijar version por defecto de MySQL
devkit services --set --postgres=15 --redis=7Fijar versiones de DB y cache
